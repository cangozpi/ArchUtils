# Heightmap Generator From Input Contour Map

Given a contour, this code generates a visual height map for terrain generation purposes.

---

## Usage:

    To install requirements:

        ```bash
        pip install requirements.txt

        ```

    * To Generate requirements file:
        ```bash
            pip freeze > requirements. txt.
        ```

---

## To run:

    * To convert _.dxf_ format to _.svg_ format:
        ```bash
            cd temp/dxf2svg
            python dxf2svg.py IsohipsTest.dxf
        ```


    ```bash
    python main.py
    ```
